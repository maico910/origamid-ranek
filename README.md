# Ranek

## Descrição do projeto
Ranek é o projeto final de curso de Vue.js do Origamid, é um sistema de e-commerce que contém:

- CRUD de Produtos
- Autenticação de Usuário
- Histórico de compra e venda
- Utilização de API

## Rodar aplicação
```
npm install
npm run serve
```
